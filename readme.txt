== Description ==

Get and show older/newer posts of the current post.

= Features =
- Get older/newer posts of the current post
- Get posts in the same category
- Show post thumbnail, post time or post author as you want

== Installation ==

1. Unzip the oldernewer-posts-widget.zip
2. Copy oldernewer-posts-widget folder to wp-content/plugins
3. Go to Plugins/Installed Plugins, find Older/Newer Posts Widget and click Active
4. Enjoy!
